<h2>Objective</h2>

Verify that the application correctly:

<ul>
  <li> Extracts Helpdesk data </li>
  <li> Handles different input formats </li>
  <li> Allows repair/item selection </li>
  <li> Generates folders and PDF files </li>
  <li> Displays correct validation messages </li>
</ul>

<h2>Scope</h2>
<h3>In Scope</h3>
<ul>
  <li> UI interactions </li>
  <li> Input validation </li>
  <li> Data extraction logic </li>
  <li> Folder creation </li>
  <li> PDF generation </li>
  <li> Popup functionality </li>
  <li> Clipboard actions </li>
</ul>
<h3>Out of Scope</h3>
<ul>
  <li> Performance testing </li>
  <li> Printer hardware compatibility </li>
  <li> Automated testing </li>
</ul>
